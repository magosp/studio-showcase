# Brand & site concepts

A small portfolio of original, **fictional** brand and website concepts. Each is a self-contained,
static site — hand-designed and built as an exercise in a distinct register.

- **DRENCH** (`/drench/`) — Cinematic — a film promo.
- **Ferrite** (`/ferrite/`) — Technical — a B2B product.
- **Liokladi** (`/liokladi/`) — A self-contained site concept.
- **MENOM** (`/menom/`) — Cinematic — a film promo.

Every brand, product, person, and studio here is invented. Nothing on these pages depicts, endorses, or
is affiliated with any real company, product, or individual.

## Structure

- `/` — landing page linking the three sites
- `/menom/`, `/ferrite/`, `/liokladi/` — each site with all of its own assets
- `.nojekyll` — served as-is (no Jekyll processing)

Static output only. Just files — no build step, no server code.
